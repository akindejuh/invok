# serverless framework (POC)

This is a simple serverless framework(runtime and cli) that can be used to run serverless functions in a durable way.


![Serverless Architecture](./asset/serverless.jpg "Architecture")

## Running

### API Controller
```sh
$ cargo run
```

Supported languages:

- [x] Golang
